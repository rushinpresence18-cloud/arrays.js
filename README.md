Project Title

JavaScript Arrays Exercise

Description

This exercise demonstrates how to work with arrays in JavaScript using colors and numbers.
It includes:

Creating arrays
Adding new items
Removing items
Printing arrays
Finding the length of arrays

Concepts Used
Arrays
Variables
.push()
.filter()
.length
console.log()

How to Run the Program
Open a code editor like Visual Studio Code
Create a JavaScript file
Copy and paste the code
Run the file using:
node filename.js

What I Learned
How to create and manage arrays in JavaScript
How to add and remove items from arrays
How to count array elements using .length
How to display array data in the console
